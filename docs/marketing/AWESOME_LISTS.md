## Submission Tips

1. **Check existing entries**: Make sure Kodra isn't already listed
2. **Follow contribution guidelines**: Each repo has different PR templates
3. **Keep it concise**: Match the style of existing entries
4. **One PR per list**: Don't spam multiple lists at once
5. **Be patient**: Maintainers are volunteers
6. **Star the repo**: Shows good faith before submitting
